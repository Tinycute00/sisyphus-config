# Standard Tools

## Core System Tools
- **Bash**: Execute terminal commands (git, npm, ls).
- **Read**: Read file contents.
- **Write**: Write file contents.
- **Edit**: String replacement in files.
- **Glob**: File pattern matching.
- **Grep**: Content search.

## Task & State Tools
- **TodoWrite**: Create/Update task list.
- **TodoRead**: Read task list.
- **Task**: Spawn sub-agents (general, explore, etc.).
- **Background_Task**: Run async agents.
- **Background_Output**: Retrieve async results.

## Knowledge & Skill Tools
- **Skill**: Load specialized workflows.
- **Memory**: Read/Write knowledge graph.
- **LSP**: Language Server Protocol (Hover, Definition, Rename).
- **Context7**: Documentation query.
- **WebSearch**: Exa AI search.

## GitHub Tools
- **gh_grep_searchGitHub**: Code search.
- **Bash (gh)**: GitHub CLI operations.
